Insights Feature Flag Misconfiguration Test

This page tests feature flag misconfiguration detection where flags are called before Insights has loaded.

Test Scenarios:

These tests simulate common feature flag misconfigurations:

  • Flag before init: Calling isFeatureEnabled() before insights.init()
  • Multiple flags before init: Several flag calls before initialization
  • Proper usage: Flags called after Insights is properly loaded

Expected result: SDK Doctor should detect feature flag misconfigurations
